다음 과정에 걸쳐 현대캐피탈의 신입 공채가 진행된다.

# 서류전형

1. 현대캐피탈에 지원하게 된 동기와 지원 직무에 필요한 역량을 기르기 위해 노력했던 경험을 기술해 주십시오. (최소 300자, 최대 500자 입력가능)
2. 실패에 대한 두려움을 극복하고 새로운 일에 도전하여 깨달음을 얻은 경험에 대해 기술해 주십시오. (최소 300자, 최대 500자 입력가능)
3. 가치관이 다른 구성원들과 협업했던 과정에서 본인의 역할은 무엇이었는지, 그 경험을 통해 어떤 역량이 향상되었는지 기술해 주십시오. (최소 300자, 최대 500자 입력가능)

자기소개서 문항은 이전 양식과 동일하기에 쉽게 찾아볼 수 있으며 위와 같다.
서류 합격 자체는 쉽게 이루어졌고 발표 또한 빨랐던 것 같다.

# 코딩 테스트

- 플랫폼 : 프로그래머스
- 문제수 : 알고리즘 4문항
- 시험 시간 : 2시간
- 화상감독 : 모니토, 검색불가, 외부IDE 불가
- 응시 직무 : BE
- 응시 가능 언어 : FE : Javascript, BE : Java, IT : C++, JS, Java, Python

백엔드로 지원하였으며 주로 Python 을 사용하여 코테를 하는데 Java 만 사용 가능했기에 살짝 긴장했다. 

문제는 총 4 문항이 나왔다.

난이도 자체는 어렵지 않았으며 마지막 4번을 제외하고는 매우 쉬웠다고 생각한다.
4번도 DP 문제인데, 골드 중상위 문제라고 생각한다.

> 역시 4일 정도 후에 발표가 났으며 합격했다.!
